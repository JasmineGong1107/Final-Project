# Final Project: Babson Notes Cloud 
## User Instruction: before running, please install...
To run codes successfully, one should install the following libraries and extension in their command prompt:
1) flask
2) wt-forms
3) flask-sqlalchemy
4) flask-bcrypt
5) flask-login
6) pillow
7) flask-mail
